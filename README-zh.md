## NNI教程

[English](./README.md) | 简体中文

+ 本项目的基础代码是用于文本分类任务(PyTorch实现)，你可以通过以下代码直接执行原始的文本分类任务：
```bash
python preprocess.py
python main.py
```

+ 在此基础上，我们展示了如何使用NNI进行自动参数调节，运行命令：
```bash
sh nni.sh
```
